插件采用poi-tl（poi template language）Word模板引擎进行渲染，当前使用版本为：1.12.2
详细的模板语法请参考链接：poi-tl模板引擎
  1. 当前可用的文档基础变量：
    1. {{title}} 设置中的标题
    2. {{author}} 设置中的作者
    3. {{date}} 当前日期
    4. {{version}} 版本号
    5. {{toc}} 版本号
  2. 当前可用的文档集合变量：
    1. {{dataSources}} 所有选中的数据库和表的数据源集合（含库和表）：
      1. das 数据源原始对象
      2. name 数据源名称
      3. comment 数据源备注
      4. dialect 数据源方言
      5. version 数据源版本
      6. namespaces 命名空间集合（在关系数据库中通常指schema）：
        1. das 命名空间原始对象
        2. name 命名空间名称
        3. comment 命名空间注释
        4. tables 数据表集合：
          1. das 数据表原始对象
          2. name 数据表名
          3. comment 数据表注释
          4. schema 数据库名
          5. primaryKey 主键列名
          6. columns 表列集合：
            1. das 列原始对象
            2. name 列名
            3. comment 注释内容
            4. type 列类型
            5. length 列长度
            6. isQuoted 是否引用
            7. defaultValue 默认值
            8. isNotNull 是否非空
            9. isPrimaryKey 是否主键
            10. isIndex 是否索引列
            11. isForeign 是否外键列
            12. isAutoGenerated 是否自动增长
            13. isComputed 是否可计算的
            14. isAuto 是否自动增长或者可计算的
            15. tableName 表名
    2. {{ds4table}} 所有选中的数据库和表的数据源集合（含库和表），
      同dataSources，方便支持LoopRowTableRenderPolicy
    3. {{ns4table}} 命名空间集合（在关系数据库中通常指schema），
      同namespaces，方便支持LoopRowTableRenderPolicy
