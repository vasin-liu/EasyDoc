说明文档：
模板采用 poi-tl（poi template language） Word模板引擎进行渲染，当前使用版本为：>poi-tl 1.12.2 poi-tl详细的模板语法请参考链接：poi-tl（poi template language）模板引擎 
    属性
    {{title}} 设置中的标题
    {{author}} 设置中的作者
    {{date}} 当前日期
    {{chineseDate}} 当前中文日期
    {{version}} 版本号
    {{toc}} 版本号

    集合
    {{dataSources}} 所有选中的数据库和表的数据源集合（含库和表）
      集合元素对象
      das 数据源原始对象
      name 数据源名称
      comment 数据源备注
      dialect 数据源方言
      version 数据源版本
      namespaces 命名空间集合（在关系数据库中通常指schema）
      集合元素对象
        das 命名空间原始对象
        name 命名空间名称
        comment 命名空间注释
        tables 数据表集合
        集合元素对象
          das 数据表原始对象
          name 数据表名
          comment 数据表注释
          schema 数据库名
          primaryKey 主键列名
          columns 表列集合
          集合元素对象
            das 列原始对象
            name 列名
            comment 注释内容
            length 长度
            isQuoted 是否引用
            defaultValue 默认值
            isNotNull 是否非空
            isPrimaryKey 是否主键
            isIndex 是否索引列
            isForeign 是否外键列
            isAutoGenerated 是否自动增长
            isComputed 是否可计算的
            isAuto 是否自动增长或者可计算的
            tableName 表名

    {{ds4table}} 所有选中的数据库和表的数据源集合（含库和表），同dataSources，方便支持LoopRowTableRenderPolicy

    {{ns4table}} 命名空间集合（在关系数据库中通常指schema），同namespaces，方便支持LoopRowTableRenderPolicy
